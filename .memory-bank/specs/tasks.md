# 実装計画

## タスク様式テンプレート

作業分解に適したパターンを選択してください：

### 主要タスクのみ
- [ ] {{NUMBER}}. {{タスク説明}}{{並列マーク}}
  - {{詳細項目1}} *(必要な場合のみ詳細を含めてください。タスクが独立している場合は、箇条書きを省略してください。)*
  - _要件: {{要件ID}}_

### 主要タスク＋サブタスク構造
- [ ] {{メインナンバー}}. {{主要タスク概要}}
- [ ] {{メインナンバー}}.{{サブナンバー}} {{サブタスク説明}}{{サブ並列マーク}}
  - {{詳細項目1}}
  - {{詳細項目2}}
  - _要件: {{要件ID}}_ *(IDのみ。説明や括弧は記載しないでください。)*

> **並列マーカー**: 並列実行できるタスクにのみ ` (P)` を付加します。`--sequential` モードで実行する場合は、マーカーを付加しません。
>
> **オプションのテストカバレッジ**: サブタスクが受け入れ基準に関連付けられた延期可能なテスト作業の場合、チェックボックスを `- [ ]*` でマークし、詳細な箇条書きで参照される要件を説明してください。
