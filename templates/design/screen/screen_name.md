# 4. イベント処理 (Actions)
## 4.1 初期表示 (OnLoad)
1. API `GET /categories` をコールし、カテゴリ選択肢を取得する。
2. ローディングを表示する。
3. 取得完了後、Selectボックスにセットする。

## 4.2 検索ボタン押下 (OnClick)
1. 入力値をチェックする。
2. API `GET /items` をパラメータ付きでコールする。
3. 結果を一覧エリアに描画する。

# 5. 状態遷移
- 正常終了時: [S-002] 一覧画面へ遷移
- エラー時: ダイアログを表示し、遷移しない
