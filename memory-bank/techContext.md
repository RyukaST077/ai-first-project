* Node.js [バージョン]
* Docker [バージョン]

### **インストール & 実行**
1. **依存関係のインストール:**
   ```bash
   pnpm install
   ```
   *注: npm や yarn は使用しないでください。*

2. **開発サーバーの起動:**
   ```bash
   pnpm dev
   ```

3. **データベースセットアップ:**
   ```bash
   docker compose up -d db
   pnpm migrate
   ```

* `.env.example` を `.env` にコピーしてください。
* **セキュリティ:** `.env` ファイルは絶対にコミットしないでください。チャットにシークレットを貼り付けないでください。
